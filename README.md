# 出口退税单证备案审核系统

一个现代化的前端应用，帮助企业自动审核出口退税单证备案资料，确保符合国家税务总局的要求。

## 🚀 功能特点

- 📄 **PDF文件上传**: 支持拖放和点击选择PDF文件
- 🔍 **智能内容解析**: 自动提取PDF中的文本内容
- ✅ **合规性审核**: 基于最新税局要求进行单证完整性检查
- 📊 **可视化结果**: 清晰展示审核结果和缺失文件
- 📋 **报告生成**: 自动生成符合税局标准的备查文件
- 📱 **响应式设计**: 完美适配移动端和桌面端

## 🎯 审核标准

系统按照国家税务总局2025年最新要求，检查以下单证：

### 必需文件
- ✅ 出口合同
- ✅ 商业发票
- ✅ 装箱单
- ✅ 运输单据（提单、空运单等）
- ✅ 报关单
- ✅ 收汇凭证

### 可选文件
- 📋 装货单
- 📋 原产地证书

## 🛠️ 技术栈

- **前端框架**: React 19 + Vite 7
- **样式系统**: CSS3 + 现代化UI设计
- **PDF处理**: 纯JavaScript文本提取
- **文件上传**: react-dropzone
- **部署**: GitHub Pages

## 📦 安装和使用

### 本地开发
```bash
# 克隆项目
git clone https://github.com/your-username/export-tax-refund-system.git

# 进入项目目录
cd export-tax-refund-system

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 访问应用
打开浏览器访问 `http://localhost:5174`

## 📋 使用说明

1. **上传文件**: 点击或拖放PDF文件到上传区域
2. **等待解析**: 系统自动解析PDF内容（通常2-3秒）
3. **查看结果**: 查看审核结果，了解已找到和缺失的文件
4. **生成报告**: 点击"生成备查文件"下载符合税局要求的审核报告
